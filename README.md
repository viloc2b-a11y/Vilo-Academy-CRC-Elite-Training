# Vilo Academy — CRC Elite Training

Bilingual (EN/ES) curriculum + demo UI for a competency-based CRC training program.

## Quick start

```bash
npm install
npm run dev
```

Open `http://localhost:3000/en` or `http://localhost:3000/es`.

## Quality checks

- **Typecheck**:

```bash
npm run typecheck
```

- **Lint**:

```bash
npm run lint
```

- **Tests**:

This repository does not yet contain an automated test suite. `npm test` fails intentionally with a clear reviewer message (see `package.json`) until a test runner is added.
