## 3) ALCOA+ Implementation Framework for Training (Execution-Layer)

ALCOA (core record integrity):

- **A — Attributable** → Unique staff/author per entry (M11)
- **L — Legible** → No white‑out; explain deletions and maintain legibility (M11)
- **C — Contemporaneous** → Same‑day completion when feasible; never backdate; document delays (M11)
- **O — Original** → Original source controls over summaries (e.g., vitals sheet > narrative summary) (M11)
- **A — Accurate** → Resolve source hierarchy conflicts without invention (M11)

ALCOA+ (system controls + completeness):

- **Complete / Consistent / Enduring / Available / Traceable** → realized via EDC controls + audit trail expectations (M12).

Training integration (how it shows up inside modules):

- **Guided Practice**: identify ALCOA+ violations and name the integrity trap.
- **Practical Exercise**: perform an ALCOA+‑compliant correction + escalation output under time pressure.
- **Answer Framework**: include short audit‑facing defense language (known vs unknown; artifact‑anchored; no invention).
